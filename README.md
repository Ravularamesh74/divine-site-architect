## 📌 Overview

**MaplePeak Staffing Platform** is a **production-grade SaaS frontend application** built for modern recruitment businesses.

It connects **top talent with organizations across Canada and the United States**, while demonstrating:

- ⚡ High-performance frontend engineering  
- 🧠 Scalable architecture  
- 🎯 Conversion-focused UI/UX  
- 🔥 Real-world SaaS patterns  

---

## ✨ Features

### 🏗️ Core Pages
- 🏠 Home (Hero, Services, Stats, FAQ, Testimonials)
- 🧠 About (Mission, Values, Team)
- 💼 Services (Detailed offerings)
- 💰 Pricing (Conversion-driven plans)
- ⭐ Testimonials (Carousel + Grid)
- 📩 Contact (Interactive form)
- ❌ 404 Page (User-friendly)

## 🚀 Getting Started

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/maplepeak.git
cd maplepeak
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Run Development Server
```bash
npm run dev
```

### 4️⃣ Build for Production
```bash
npm run build
```
